# Code of Conduct

## Our Commitment

We want NegotiationForge to be an open, practical, and respectful community.
Contributors, maintainers, and users should be able to participate without harassment, hostility, or exclusion.

## Expected Behavior

- Be respectful and direct.
- Critique ideas, code, and decisions rather than people.
- Keep technical discussions grounded in evidence.
- Welcome good-faith questions from new contributors.
- Provide actionable feedback when reporting problems.

## Unacceptable Behavior

- Harassment, personal attacks, or intimidation
- Discriminatory or hateful language
- Deliberate trolling or disruption
- Publishing private information without permission
- Repeated bad-faith arguments or abusive issue / PR behavior

## Enforcement

Project maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior violates this code of conduct.

## Scope

This code of conduct applies to:

- GitHub issues
- pull requests
- discussions
- project documentation
- other public community spaces related to this repository

## Reporting

If you encounter unacceptable behavior, open a private channel with the maintainers or create an issue requesting maintainer contact.

## Notes

This document is intentionally lightweight. The goal is practical collaboration, not bureaucratic process.
